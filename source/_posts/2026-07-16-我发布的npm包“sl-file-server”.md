这时我很久之前做的包，`sl-file-server`，用来在网页启动文件管理器，但只能浏览

我做这个包是用来解决在 serverless 环境下展示项目的文件路径的

我拿来在 eo 上搭用于浏览我的博客插入图片，虽然我很少在博客里插入图片

安装

```npm
npm i -g sl-file-server
```

使用
```sh
sl-file-server
```

```sh
sl-file-server --sl
```
