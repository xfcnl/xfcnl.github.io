# XF

[github 线路](https://xfcnl.github.io)　[cloudflare 线路](https://husd.cc.cd)　[edgeone 线路](https://blog.sfvg.de5.net)

基于 [Hexo](https://hexo.io) 构建的自用博客，主题为自制的 `xf`

## 本地预览

```bash
# 安装依赖
npm install

# 启动开发服务器
npx hexo server

# 生成静态文件
npx hexo generate
```

浏览器打开 `http://localhost:4000` 即可预览

## 目录结构

```
source/_posts/  — 博客文章
source/_data/   — 配置数据（社交链接、友链等）
themes/xf/      — 自制主题
```
